Support

Support is available through the following channels:
  • Discussion Board - The board is the primary support channel. Please search the board before asking a question, maybe your question has already been answered.
  • Issue tracking - Use this channel to report bugs and request enhancements. It is recommended to discuss an enhancement/bug at the discussion board before posting it to the issue tracking system.
  • E-Mail - This is a "default" channel as in switch-case, i.e. use it if the above channels cannot be used for some reason.
